<!-- GETTING STARTED -->
## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/owengretzinger/habits-together.git
   ```
2. Navigate to the mobile app folder
   ```sh
   cd mobile-app
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the code
   ```sh
   npm start
   ```
5. Download the Expo Go app on your phone
   https://expo.dev/go
6. Scan the QR code that shows up in the terminal

For more about setting up a development environment, check the Expo docs: https://docs.expo.dev/get-started/set-up-your-environment
